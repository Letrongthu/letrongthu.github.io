Kho dữ liệu

Tải dữ liệu lịch sử & realtime (nếu có quyền), chuẩn hóa cột, timezone, định dạng số. Bạn có thể thay phần này bằng bảng/iframe thật.

Triển khai nhanh trên GitHub Pages

  1. Tạo repo letrongthu.github.io (Public).
  2. Tải lên index.html và thư mục assets/.
  3. Settings → Pages → Source: Deploy from a branch → main / /root.
  4. Chờ build xong, mở letrongthu.github.io.

Mẹo: Link CSS/JS/ảnh nên là đường dẫn tương đối (ví dụ ./assets/css/style.css).

Muốn mình nhúng gói Readdy.ai vào site này?

Gửi gói .zip hoặc link publish, mình sẽ chuyển thành mã tĩnh và tối ưu đường dẫn cho GitHub Pages.
